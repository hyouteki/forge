## Forge
Simple 3D game engine in C++ with OpenGL.

## Quick Start
### GNU/Linux
``` bash
chmod +x build.sh clean.sh
./build.sh
```
### Windows
``` bash
build
```

## Dependencies
- [CMake](https://cmake.org/)
- [OpenGL](https://www.opengl.org/)
- [GLFW](https://www.glfw.org/)
- [GLEW](https://glew.sourceforge.net/)
